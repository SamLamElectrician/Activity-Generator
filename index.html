💖Welcome to Sam and Phil's Date Generator 💖

This app will suggest a date activity based on the parameters you enter.

  • Type in where you are based in the city input to get the current weather of the day
  • Select the type of activity you would like to partake in, in our drop down menu
  • Select on the slider how expensive you would like the activity to be
  • Click generate for your suggestion
  • If you'd like to randomize all the parameters for a suprise just hit the Randomize button
